System Features Search

Search

Description

      All modules in the system will utilize a form of search and there will be varying search criteria for each form. The search feature will utilize tool strips which would allow the user to input their criteria and when they input their criteria, they would click the search button and all records related to the search criteria entered by the user will be displayed.

Functional Requirements

  1. The user must be able to search for records without having to enter the full information related to the record.
  2. The user must be able to sort the records in the table in any order they wish.
  3. The user must be able to navigate to other forms from the search forms.

Non-Functional Requirements

  1. The user must be able to view all records in the table before a search is applied.
